# 來料品質異常迴歸分析｜AI 提示詞工具

> **數據分析與企業決策課程臨摹案例**  
> 羅懷鈞教授「數據分析與企業決策」｜100 批次樣本數據｜OLS 多元迴歸

---

## 🎯 這是什麼？

這是一個**免安裝、開源、可複製**的 AI 提示詞工具網頁，幫助管理者與學習者透過迴歸分析，將來料品質異常問題拆解成「可量化的決策槓桿」。

網頁分為兩大區塊：
- **首頁**：三組 AI 提示詞，複製貼上即可讓 Kimi / ChatGPT / Gemini 自動完成完整分析
- **次頁**：100 批次練習數據，一鍵複製 CSV 格式，貼上即可練習

## 📋 三組提示詞一覽

| 提示詞 | 用途 | 適合對象 |
|---|---|---|
| **提示詞 01**｜完整分析流程 | 六步驟：問題定義 → 敘述統計 → 相關分析 → 迴歸建模 → 情境模擬 → 決策建議 | 想跑完整分析的人 |
| **提示詞 02**｜數據解讀與圖表 | 自動生成散點圖、箱型圖、熱力圖 | 需要視覺化呈現的人 |
| **提示詞 03**｜決策簡報生成 | 產出給老闆看的簡報結構（選項A/B/C + 行動計畫） | 需要做管理簡報的人 |

## 🧮 迴歸模型摘要

```
Y = 24.57 − 0.20·X₁ − 0.17·X₂ + 0.04·X₃ − 0.02·X₄ − 0.12·X₅

Y  = 來料品質異常率(%)
X₁ = 物料精密等級 (1–10分)
X₂ = 供應商自檢合格率 (%)
X₃ = 加急訂單佔比 (%)
X₄ = 進料檢驗抽樣比例 (%)
X₅ = 檢驗人員技能熟練度 (1–10分)

R² = 0.721（調整後 R² = 0.706）｜n = 100 批次
```

**影響力排序**（標準化係數 |β|）：
1. 供應商自檢合格率 (0.66) 🥇
2. 物料精密等級 (0.30) 🥈
3. 加急訂單佔比 (0.34) 🥉
4. 檢驗人員熟練度 (0.20)
5. 進料抽樣比例 (0.14)
